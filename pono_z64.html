Zaokrąglanie narożników oraz centrowanie DIVów

1) Tabela zaokrągleń narożników obszarów bloków (np. DIV)

Zaokrąglenie narożnika Selektor CSS
Górny lewy border-top-left-radius: wartość;
Górny prawy border-top-right-radius: wartość;
Dolny prawy border-bottom-right-radius: wartość;
Dolny lewy border-bottom-left-radius: wartość;

2) Zastosowanie i uwagi dotyczące zaokrąglania narożników

Zaokrąglanie narożników stosuje się w celu poprawy estetyki elementów na stronie. Można je stosować do dowolnych znaczników, takich jak:

  • p — akapit
  • h1 — nagłówek
  • div — blok

Wartość zaokrąglenia może być podana w pikselach, centymetrach lub procentach. Możliwe jest:

  • podanie jednej wartości — zaokrąglenie symetryczne,
  • podanie dwóch wartości — pierwsza określa promień poziomy, druga pionowy.

Element nie musi mieć obramowania, aby zaokrąglenie było widoczne — wystarczy tło. Należy jednak uważać, aby promień nie był zbyt duży, bo tekst może wyjść poza obszar.

3) Przykład z instrukcją

div {
    width: 200px;
    height: 100px;
    background-color: lightblue;
    border-top-right-radius: 20px;
}

Powyższy kod tworzy prostokąt z zaokrąglonym prawym górnym narożnikiem.

4) Centrowanie DIVów

Aby wycentrować DIV poziomo, stosujemy: 

div {
    width: 300px;
    margin-left: auto;
    margin-right: auto;
}

Warunek: DIV musi mieć ustawioną szerokość. Wtedy przeglądarka automatycznie rozdziela wolne miejsce po bokach.

Aby wycentrować DIV pionowo, stosuje się: 

div {
    position: absolute;
    top: 50%;
    transform: translateY(-50%);
}

lub nowocześnie: 

body {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
}
